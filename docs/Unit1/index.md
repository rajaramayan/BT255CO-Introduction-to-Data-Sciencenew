# Unit 1 — Introduction to Data Science
**Duration: 4 Hours**

---

## Overview

This unit lays the foundation for the entire course. We start by understanding what Data Science is, where it is used, and where its boundaries lie. We then study the different types of data, the standard data science lifecycle, and two widely used methodologies — CRISP-DM and OSEMN.

## Unit Learning Objectives

By the end of Unit 1, you will be able to:

- [ ] Define Data Science and explain its interdisciplinary scope
- [ ] List real-world applications across industries
- [ ] Identify limitations and ethical concerns
- [ ] Classify data into categorical, binomial, nominal, and ordinal types
- [ ] Describe the stages of the Data Science lifecycle
- [ ] Explain data collection, cleaning, and preprocessing steps
- [ ] Describe data storage options and retrieval methods
- [ ] Compare CRISP-DM and OSEMN methodologies

## Quick Reference

| Term | Meaning |
|------|---------|
| Data Science | Interdisciplinary field: statistics + programming + domain knowledge |
| Categorical Data | Data that represents categories or groups |
| Binomial Data | Data with exactly two possible outcomes (Yes/No) |
| Nominal Data | Categories with no inherent order |
| Ordinal Data | Categories with a meaningful order |
| CRISP-DM | Cross-Industry Standard Process for Data Mining (6 phases) |
| OSEMN | Obtain, Scrub, Explore, Model, iNterpret |
| Preprocessing | Cleaning, encoding, and transforming raw data |

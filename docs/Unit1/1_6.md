# 1.6 Collection, Cleaning, and Preprocessing

---

## Theory

### Data Collection

Data can be obtained from many sources. The method of collection affects quality, cost, and legality.

| Source | Method | Example |
|--------|--------|---------|
| **Databases** | SQL queries | Customer records in MySQL |
| **CSV / Excel** | `pd.read_csv()` | Kaggle dataset downloads |
| **APIs** | HTTP requests | Weather API, Twitter API |
| **Web Scraping** | BeautifulSoup, Scrapy | News articles, e-commerce prices |
| **Sensors / IoT** | Real-time streams | Temperature sensors, GPS |
| **Surveys** | Google Forms, Kobo | Primary research data |
| **Social Media** | Platform APIs | Tweets, Instagram posts |

---

### Data Cleaning

Data cleaning (also called **data cleansing** or **data scrubbing**) is the process of detecting and correcting (or removing) inaccurate records from a dataset.

#### Common Cleaning Tasks

| Task | Technique |
|------|-----------|
| **Missing values** | Drop rows, impute with mean/median/mode, or use ML imputation |
| **Duplicates** | `df.drop_duplicates()` |
| **Outliers** | Z-score, IQR method, domain knowledge |
| **Inconsistent formats** | `str.strip()`, `str.lower()`, date parsing |
| **Wrong data types** | `df["col"].astype(int)` |
| **Noise** | Smoothing (rolling average), removing measurement errors |

### Data Preprocessing

Preprocessing transforms raw cleaned data into a format suitable for ML models.

| Step | Purpose | Technique |
|------|---------|-----------|
| **Encoding** | Convert categorical to numbers | One-Hot, Label Encoding |
| **Scaling** | Bring features to same range | Min-Max, Standardisation (Z-score) |
| **Splitting** | Separate training/test data | `train_test_split` |
| **Balancing** | Handle class imbalance | SMOTE, undersampling |
| **Normalisation** | Scale values to [0, 1] | Min-Max Scaler |
| **Standardisation** | Mean=0, Std=1 | Standard Scaler |

#### Min-Max Normalisation

$$
x_{norm} = \frac{x - x_{min}}{x_{max} - x_{min}}
$$

#### Z-Score Standardisation

$$
z = \frac{x - \mu}{\sigma}
$$

where $\mu$ is the mean and $\sigma$ is the standard deviation.

### Python Program — Full Cleaning & Preprocessing Pipeline

```python linenums="1" title="cleaning_preprocessing.py"
# Program : Data Cleaning and Preprocessing Pipeline
# Topic   : 1.6 Collection, Cleaning, and Preprocessing
# Author  : BT255CO Lecture Notes

import pandas as pd
import numpy as np
from sklearn.preprocessing import MinMaxScaler, LabelEncoder

# -------------------------------------------------------
# Step 1: Simulated raw dataset (with common issues)
# -------------------------------------------------------
data = {
    "name":    ["Alice", "Bob", "Carol", "Alice", "Eve", None],
    "age":     [25, 200, 30, 25, 28, 22],
    "salary":  [50000, None, 60000, 50000, 45000, 52000],
    "city":    ["Delhi", "mumbai", "DELHI", "Delhi", "Mumbai", None],
    "dept":    ["HR", "IT", "Finance", "HR", "IT", "Finance"],
    "rating":  [4, 3, 5, 4, None, 2],
}
df = pd.DataFrame(data)
print("Raw Data:")
print(df)
print()

# -------------------------------------------------------
# Step 2: Remove duplicates
# -------------------------------------------------------
df = df.drop_duplicates()
print(f"After removing duplicates: {len(df)} rows\n")

# -------------------------------------------------------
# Step 3: Fix inconsistent text
# -------------------------------------------------------
df["city"] = df["city"].str.strip().str.title()
print(f"Cities normalised: {df['city'].unique()}\n")

# -------------------------------------------------------
# Step 4: Fix outliers in 'age' (valid: 18–80)
# -------------------------------------------------------
df.loc[df["age"] > 80, "age"] = np.nan      # mark as missing

# -------------------------------------------------------
# Step 5: Impute missing values
# -------------------------------------------------------
df["salary"].fillna(df["salary"].median(), inplace=True)  # median for salary
df["rating"].fillna(df["rating"].mean(),   inplace=True)  # mean for rating
df["age"].fillna(df["age"].median(),       inplace=True)  # median for age
df["city"].fillna("Unknown",               inplace=True)  # placeholder for city
df["name"].fillna("Unknown",               inplace=True)

print("After Imputation:")
print(df)
print()

# -------------------------------------------------------
# Step 6: Encode categorical column 'dept'
# -------------------------------------------------------
le = LabelEncoder()
df["dept_encoded"] = le.fit_transform(df["dept"])
print("Label Encoded 'dept':")
print(dict(zip(le.classes_, le.transform(le.classes_))))
print()

# -------------------------------------------------------
# Step 7: Scale numeric features (salary, age)
# -------------------------------------------------------
scaler = MinMaxScaler()
df[["salary_scaled", "age_scaled"]] = scaler.fit_transform(df[["salary", "age"]])

print("Scaled Features (salary, age → 0–1 range):")
print(df[["salary", "salary_scaled", "age", "age_scaled"]].round(3))
```

## Summary

!!! success "Key Takeaways"
    - Data collection sources include databases, APIs, files, sensors, and scraping
    - Data cleaning handles missing values, duplicates, outliers, and format inconsistencies
    - Preprocessing encodes, scales, and splits data for use in ML models
    - **Min-Max Scaling** normalises to [0, 1]; **Z-Score** standardises to mean=0, std=1
    - Always impute missing values **before** scaling, and scale **after** encoding

# 2.3 Cleaning and Standardizing Data

---

## Theory

After identifying data quality issues, the next step is to **clean and standardise** the data. Standardisation ensures that all values follow a consistent format so they can be compared, aggregated, and fed into models reliably.

---

### Cleaning vs. Standardisation

| Concept | Focus | Example |
|---------|-------|---------|
| **Cleaning** | Remove errors, outliers, inconsistencies | Replace age=200 with NaN |
| **Standardisation** | Unify format and representation | Convert all dates to "YYYY-MM-DD" |

---

### Outlier Detection Methods

#### Method 1 — Z-Score

A data point is considered an outlier if its Z-score is beyond ±3:

$$
z = \frac{x - \mu}{\sigma}
$$

#### Method 2 — IQR (Interquartile Range)

$$
\text{Lower bound} = Q_1 - 1.5 \times IQR
$$
$$
\text{Upper bound} = Q_3 + 1.5 \times IQR
$$

where $IQR = Q_3 - Q_1$.

---

### Standardisation Techniques

| Type | Python Tool | Output |
|------|-------------|--------|
| **Text Case** | `str.lower()`, `str.title()` | Uniform casing |
| **Date Format** | `pd.to_datetime(format=...)` | ISO 8601 dates |
| **Numeric Scaling** | `MinMaxScaler`, `StandardScaler` | [0,1] or mean=0 |
| **String Trim** | `str.strip()` | Remove whitespace |
| **Category Codes** | `str.replace()`, mapping dict | Consistent labels |

### Python Program — Comprehensive Cleaning Pipeline

```python linenums="1" title="cleaning_pipeline.py"
# Program : Data Cleaning and Standardisation Pipeline
# Topic   : 2.3 Cleaning and Standardizing Data
# Author  : BT255CO Lecture Notes

import pandas as pd
import numpy as np
from scipy import stats

# -------------------------------------------------------
# Raw dataset
# -------------------------------------------------------
np.random.seed(42)
data = {
    "name":        ["  Alice ", "BOB", "carol", "DAVID", "  Eve"],
    "age":         [25, 30, 150, 28, 22],          # 150 = outlier
    "salary":      [50000, 120000, 60000, 200000, 45000],  # 200000 = outlier
    "join_date":   ["2023/01/15", "15-06-2022", "2022.08.20",
                    "Nov 10 2021", "2023-03-01"],
    "gender":      ["F", "Male", "female", "M", "f"],
    "score":       [85, 72, 90, 68, None],
}

df = pd.DataFrame(data)
print("Original Data:")
print(df)
print()

# =========================================================
# Step 1: Strip whitespace and standardise name case
# =========================================================
df["name"] = df["name"].str.strip().str.title()
print(f"Names cleaned: {df['name'].tolist()}\n")

# =========================================================
# Step 2: Standardise gender
# =========================================================
gender_map = {
    "f": "Female", "female": "Female",
    "m": "Male",   "male":   "Male"
}
df["gender"] = df["gender"].str.lower().map(gender_map)
print(f"Gender standardised: {df['gender'].tolist()}\n")

# =========================================================
# Step 3: Parse all date formats to datetime
# =========================================================
df["join_date"] = pd.to_datetime(df["join_date"], infer_datetime_format=True)
print(f"Dates standardised:\n{df['join_date'].dt.strftime('%Y-%m-%d').tolist()}\n")

# =========================================================
# Step 4: Detect outliers using IQR for 'age'
# =========================================================
Q1  = df["age"].quantile(0.25)
Q3  = df["age"].quantile(0.75)
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

print(f"Age IQR Analysis: Q1={Q1}, Q3={Q3}, IQR={IQR}")
print(f"  Valid range: [{lower_bound:.1f}, {upper_bound:.1f}]")
outlier_mask = (df["age"] < lower_bound) | (df["age"] > upper_bound)
print(f"  Outliers: {df.loc[outlier_mask, 'age'].tolist()}")

# Replace outlier age with median
df.loc[outlier_mask, "age"] = df["age"].median()
print(f"  Age after fix: {df['age'].tolist()}\n")

# =========================================================
# Step 5: Detect salary outliers using Z-score
# =========================================================
z_scores = np.abs(stats.zscore(df["salary"]))
salary_outliers = df[z_scores > 1.5]   # threshold of 1.5 (small sample)
print(f"Salary Z-score outliers:\n{salary_outliers[['name', 'salary']]}")

# Cap outliers at upper limit (Winsorisation)
upper_cap = df["salary"].quantile(0.95)
df["salary"] = df["salary"].clip(upper=upper_cap)
print(f"Salary after capping at ₹{upper_cap:,.0f}: {df['salary'].tolist()}\n")

# =========================================================
# Step 6: Impute missing score
# =========================================================
df["score"].fillna(df["score"].mean(), inplace=True)
print(f"Score after imputation: {df['score'].tolist()}\n")

# =========================================================
# Final clean dataset
# =========================================================
print("Final Clean Dataset:")
print(df[["name", "age", "salary", "gender", "join_date", "score"]])
```

## Summary

!!! success "Key Takeaways"
    - Cleaning fixes errors; standardisation unifies format
    - Outliers can be detected with the **IQR method** or **Z-score method**
    - **Winsorisation** (clipping) is preferred over deletion when outliers may be real extreme values
    - Text standardisation: `strip()` → `lower()` → `map(dict)` → done
    - Always keep a copy of the raw data before applying any cleaning transformations

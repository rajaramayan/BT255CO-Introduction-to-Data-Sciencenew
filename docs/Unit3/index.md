# Unit 3 — Statistical Analysis in Python
**Duration: 3 Hours**

## Overview

Statistics forms the mathematical backbone of Data Science. This unit covers two essential statistical techniques: **hypothesis testing** — the formal framework for drawing conclusions from data — and **correlation and regression analysis** — the tools for measuring and modelling relationships between variables.

## Quick Reference

| Term | Meaning |
|------|---------|
| Hypothesis Testing | Statistical procedure to make decisions about a population |
| Null Hypothesis (H₀) | The default assumption — no effect or no difference |
| Alternative Hypothesis (H₁) | The claim being tested |
| p-value | Probability of observing results this extreme if H₀ is true |
| α (alpha) | Significance level — usually 0.05 |
| t-test | Test comparing means of one or two groups |
| Chi-Square Test | Test for association between categorical variables |
| Correlation | Measure of linear relationship between two variables (−1 to +1) |
| Regression | Model that predicts a continuous variable from one or more predictors |

# 4.2 Data Visualization with Matplotlib and Seaborn

---

## Theory

Data visualisation converts numbers into pictures that reveal patterns, trends, and anomalies that are invisible in raw tables.

### Matplotlib vs. Seaborn

| Feature | Matplotlib | Seaborn |
|---------|-----------|---------|
| **Level** | Low-level, fine-grained control | High-level, concise syntax |
| **Statistical plots** | Manual | Built-in (boxplot, violin, heatmap) |
| **Default style** | Basic | Aesthetically polished |
| **Relationship to other** | Foundation | Built on top of Matplotlib |
| **Best for** | Custom, precise figures | Quick statistical exploration |

### Choosing the Right Chart

| Goal | Chart Type |
|------|-----------|
| Distribution of one variable | Histogram, KDE plot, Box plot |
| Relationship between two variables | Scatter plot, Line plot |
| Comparing categories | Bar chart, Box plot, Violin plot |
| Correlation matrix | Heatmap |
| Composition | Pie chart, Stacked bar |
| Time series | Line plot |

### Python Program — Complete Visualization Gallery

```python linenums="1" title="visualization_gallery.py"
# Program : Data Visualization with Matplotlib and Seaborn
# Topic   : 4.2 Visualization
# Author  : BT255CO Lecture Notes

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

np.random.seed(42)
sns.set_theme(style="whitegrid")

# -------------------------------------------------------
# Dataset: student exam performance
# -------------------------------------------------------
n = 150
data = {
    "study_hours": np.random.uniform(1, 10, n),
    "sleep_hours": np.random.uniform(4, 9, n),
    "score":       None,
    "branch":      np.random.choice(["CSE", "ECE", "Mech", "Civil"], n),
    "gender":      np.random.choice(["Male", "Female"], n),
}
data["score"] = (8 * data["study_hours"] + 3 * data["sleep_hours"]
                  + np.random.normal(0, 5, n)).clip(0, 100)
df = pd.DataFrame(data)

fig, axes = plt.subplots(2, 3, figsize=(15, 10))
fig.suptitle("Student Exam Performance — Visualization Gallery", fontsize=15)

# -------------------------------------------------------
# Plot 1: Histogram — distribution of scores
# -------------------------------------------------------
axes[0, 0].hist(df["score"], bins=20, color="#5c6bc0", edgecolor="white")
axes[0, 0].set_title("Histogram — Score Distribution")
axes[0, 0].set_xlabel("Score")
axes[0, 0].set_ylabel("Frequency")
axes[0, 0].axvline(df["score"].mean(), color="red",
                    linestyle="--", label=f"Mean={df['score'].mean():.1f}")
axes[0, 0].legend()

# -------------------------------------------------------
# Plot 2: Box Plot — scores by branch
# -------------------------------------------------------
sns.boxplot(data=df, x="branch", y="score",
            palette="Set2", ax=axes[0, 1])
axes[0, 1].set_title("Box Plot — Score by Branch")

# -------------------------------------------------------
# Plot 3: Scatter Plot — study hours vs score
# -------------------------------------------------------
axes[0, 2].scatter(df["study_hours"], df["score"],
                    alpha=0.5, c="#3f51b5", edgecolors="none")
m, b = np.polyfit(df["study_hours"], df["score"], 1)
x_line = np.linspace(1, 10, 100)
axes[0, 2].plot(x_line, m * x_line + b, "r--", label="Trend")
axes[0, 2].set_title("Scatter — Study Hours vs Score")
axes[0, 2].set_xlabel("Study Hours")
axes[0, 2].set_ylabel("Score")
axes[0, 2].legend()

# -------------------------------------------------------
# Plot 4: Bar Chart — mean score per branch
# -------------------------------------------------------
branch_means = df.groupby("branch")["score"].mean().reset_index()
axes[1, 0].bar(branch_means["branch"], branch_means["score"],
                color=["#5c6bc0", "#7986cb", "#3f51b5", "#283593"])
axes[1, 0].set_title("Bar Chart — Average Score by Branch")
axes[1, 0].set_ylabel("Mean Score")

# -------------------------------------------------------
# Plot 5: Heatmap — correlation matrix
# -------------------------------------------------------
corr = df[["study_hours", "sleep_hours", "score"]].corr()
sns.heatmap(corr, annot=True, fmt=".2f", cmap="Blues",
            ax=axes[1, 1], square=True)
axes[1, 1].set_title("Heatmap — Correlation Matrix")

# -------------------------------------------------------
# Plot 6: Violin Plot — score by gender
# -------------------------------------------------------
sns.violinplot(data=df, x="gender", y="score",
               palette="muted", ax=axes[1, 2])
axes[1, 2].set_title("Violin Plot — Score by Gender")

plt.tight_layout()
plt.savefig("../images/visualization_gallery.png", dpi=150, bbox_inches="tight")
plt.show()
print("Plot saved to docs/images/visualization_gallery.png")
```

### Chart Reference Guide

=== "Histogram"
    ```python
    plt.hist(data, bins=20, color="steelblue", edgecolor="white")
    plt.xlabel("Value")
    plt.ylabel("Frequency")
    plt.title("Distribution")
    ```
    **Use:** Show the frequency distribution of a single continuous variable.

=== "Box Plot"
    ```python
    sns.boxplot(data=df, x="category", y="value")
    ```
    **Use:** Show median, quartiles, and outliers. Compare distributions across groups.

=== "Scatter Plot"
    ```python
    plt.scatter(x, y, alpha=0.5)
    ```
    **Use:** Visualise the relationship between two continuous variables.

=== "Bar Chart"
    ```python
    sns.barplot(data=df, x="category", y="value")
    ```
    **Use:** Compare aggregate values across categories.

=== "Heatmap"
    ```python
    sns.heatmap(df.corr(), annot=True, cmap="Blues")
    ```
    **Use:** Visualise a correlation matrix or any 2D grid of values.

=== "Line Plot"
    ```python
    plt.plot(x, y, marker="o")
    ```
    **Use:** Show trends over time or ordered sequences.

=== "Violin Plot"
    ```python
    sns.violinplot(data=df, x="category", y="value")
    ```
    **Use:** Show distribution shape + summary statistics (combines box + KDE).

## Summary

!!! success "Key Takeaways"
    - **Matplotlib** provides low-level, precise control; **Seaborn** simplifies statistical plotting
    - Histograms show **distribution**; box plots show **spread + outliers**; scatter plots show **relationships**
    - **Heatmaps** visualise correlation matrices at a glance
    - **Violin plots** show both the distribution shape and summary statistics
    - Always label axes, add a title, and choose colours purposefully

# 6.2.2 Decision Trees and Random Forests

---

## Theory

### Decision Trees

!!! note "Definition"
    A **Decision Tree** is a supervised learning algorithm that splits the data into subsets by asking a series of **yes/no questions** about the features, creating a tree-like flowchart where each leaf represents a prediction.

**Structure:**
- **Root Node** — the first split (most informative feature)
- **Internal Nodes** — intermediate decision points
- **Leaf Nodes** — final predictions (class label or value)

**Splitting criteria:**
- **Gini Impurity** (used by CART): $Gini = 1 - \sum p_i^2$
- **Information Gain / Entropy**: $H = -\sum p_i \log_2 p_i$

**Limitations:**
- Prone to **overfitting** when trees are deep
- **High variance** — small changes in data produce very different trees

---

### Random Forests

!!! note "Definition"
    A **Random Forest** is an **ensemble** method that builds many independent Decision Trees and combines their predictions by **majority voting** (classification) or **averaging** (regression).

**Key ideas:**
1. **Bootstrap Aggregating (Bagging)** — each tree is trained on a random sample (with replacement) of the training data
2. **Feature Randomness** — each split considers only a random subset of features (reduces correlation between trees)

$$
\hat{y} = \text{mode}\{f_1(x), f_2(x), \ldots, f_T(x)\}  \quad \text{(classification)}
$$

$$
\hat{y} = \frac{1}{T}\sum_{t=1}^{T} f_t(x)  \quad \text{(regression)}
$$

**Advantages over single Decision Tree:**
- Reduced overfitting (variance reduction by averaging)
- More robust and accurate
- Provides **feature importance** scores

### Python Program — Decision Tree and Random Forest

```python linenums="1" title="decision_tree_rf.py"
# Program : Decision Trees and Random Forests
# Topic   : 6.2.2 Decision Trees and Random Forests
# Author  : BT255CO Lecture Notes

import numpy as np
import pandas as pd
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.tree import DecisionTreeClassifier, export_text
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import (accuracy_score, classification_report,
                              confusion_matrix)
from sklearn.preprocessing import StandardScaler

# -------------------------------------------------------
# Load dataset
# -------------------------------------------------------
iris = load_iris()
X, y = iris.data, iris.target
feature_names = iris.feature_names
class_names   = iris.target_names

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42, stratify=y
)

print(f"Dataset: {len(X)} samples, {X.shape[1]} features, {len(class_names)} classes")
print(f"Train: {len(X_train)}  |  Test: {len(X_test)}\n")

# =========================================================
# PART 1: Decision Tree
# =========================================================
print("=" * 55)
print("PART 1: Decision Tree Classifier")
print("=" * 55)

dt = DecisionTreeClassifier(max_depth=3, criterion="gini", random_state=42)
dt.fit(X_train, y_train)
y_pred_dt = dt.predict(X_test)

print(f"Max depth: 3 | Criterion: Gini")
print(f"Training Accuracy : {dt.score(X_train, y_train) * 100:.1f}%")
print(f"Test Accuracy     : {accuracy_score(y_test, y_pred_dt) * 100:.1f}%")

# Print the tree structure
print("\nDecision Tree Rules:")
print(export_text(dt, feature_names=feature_names))

print("Feature Importances (Decision Tree):")
for feat, imp in zip(feature_names, dt.feature_importances_):
    bar = "█" * int(imp * 30)
    print(f"  {feat:<25} {imp:.3f} {bar}")

# =========================================================
# PART 2: Random Forest
# =========================================================
print("\n" + "=" * 55)
print("PART 2: Random Forest Classifier")
print("=" * 55)

rf = RandomForestClassifier(n_estimators=100, max_depth=5,
                              random_state=42, n_jobs=-1)
rf.fit(X_train, y_train)
y_pred_rf = rf.predict(X_test)

print(f"Trees: 100 | Max depth: 5")
print(f"Training Accuracy : {rf.score(X_train, y_train) * 100:.1f}%")
print(f"Test Accuracy     : {accuracy_score(y_test, y_pred_rf) * 100:.1f}%")

# Cross-validation
cv_scores = cross_val_score(rf, X, y, cv=5, scoring="accuracy")
print(f"5-Fold CV Accuracy: {cv_scores.mean()*100:.1f}% ± {cv_scores.std()*100:.1f}%")

print("\nFeature Importances (Random Forest):")
for feat, imp in sorted(zip(feature_names, rf.feature_importances_),
                         key=lambda x: -x[1]):
    bar = "█" * int(imp * 40)
    print(f"  {feat:<25} {imp:.3f} {bar}")

print("\nClassification Report (Random Forest):")
print(classification_report(y_test, y_pred_rf, target_names=class_names))

# =========================================================
# PART 3: Overfitting Demo — Depth vs. Accuracy
# =========================================================
print("=" * 55)
print("PART 3: Decision Tree Depth vs. Accuracy")
print("=" * 55)
print(f"{'Depth':<8} {'Train Acc':<12} {'Test Acc':<12} {'Overfit?'}")
print("-" * 45)
for depth in range(1, 11):
    dt_d = DecisionTreeClassifier(max_depth=depth, random_state=42)
    dt_d.fit(X_train, y_train)
    tr_acc = dt_d.score(X_train, y_train)
    te_acc = dt_d.score(X_test,  y_test)
    overfit = "YES ⚠" if tr_acc - te_acc > 0.05 else "No"
    print(f"{depth:<8} {tr_acc*100:<12.1f} {te_acc*100:<12.1f} {overfit}")
```

## Summary

!!! success "Key Takeaways"
    - Decision Trees are **interpretable** but prone to **overfitting** at high depths
    - **Random Forests** build many trees, average their predictions, and dramatically reduce overfitting
    - `max_depth` is the most important hyperparameter for controlling tree complexity
    - **Feature importance** from Random Forests reveals which features drive predictions
    - 5-fold **cross-validation** gives a more reliable accuracy estimate than a single train/test split

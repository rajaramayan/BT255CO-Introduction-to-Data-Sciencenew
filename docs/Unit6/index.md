# Unit 6 — Machine Learning
**Duration: 10 Hours**

---

## Overview

Machine Learning (ML) is the core of modern Data Science. Instead of being explicitly programmed with rules, ML systems **learn patterns from data** and use those patterns to make predictions or decisions. This is the largest unit in the course, covering theory, key algorithms, and hands-on Python implementation.

## Unit Learning Objectives

- [ ] Distinguish between supervised, unsupervised, and reinforcement learning
- [ ] Implement Linear Regression and interpret coefficients
- [ ] Build Decision Tree and Random Forest classifiers
- [ ] Apply SVM for binary classification
- [ ] Apply K-Means and hierarchical clustering
- [ ] Reduce dimensionality using PCA and interpret components

## Quick Reference

| Algorithm | Type | Task | Key Metric |
|-----------|------|------|-----------|
| Linear Regression | Supervised | Regression | R², RMSE |
| Logistic Regression | Supervised | Classification | Accuracy, F1 |
| Decision Tree | Supervised | Both | Accuracy, Gini |
| Random Forest | Supervised | Both | Accuracy, Feature Importance |
| SVM | Supervised | Both | Accuracy, Kernel |
| K-Means | Unsupervised | Clustering | Inertia, Silhouette Score |
| Hierarchical | Unsupervised | Clustering | Dendrogram |
| PCA | Unsupervised | Dimensionality Reduction | Explained Variance |

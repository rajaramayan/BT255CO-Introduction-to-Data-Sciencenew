# Unit 5 — Data Visualization
**Duration: 4 Hours**

---

## Overview

Effective data visualisation is a core skill of every data scientist. This unit goes beyond basic charts to explore **comprehensive visualisation strategies** using graphs and charts for different data scenarios, and **interactive visualisation tools** — Plotly, Google Charts, and Leaflet — that let users explore data dynamically.

## Unit Learning Objectives

- [ ] Select the appropriate chart type for a given data scenario
- [ ] Create multi-variable visualisations to compare groups and trends
- [ ] Plot time-series data with proper axis formatting
- [ ] Build interactive charts using Plotly Express
- [ ] Describe the capabilities of Google Charts and Leaflet.js

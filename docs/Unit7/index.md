# Unit 7 — Introduction to Big Data
**Duration: 5 Hours**

---

## Overview

When datasets become too large, too fast, or too varied for traditional tools to handle, we enter the domain of **Big Data**. This unit covers the history, characteristics, challenges, technologies, and real-world applications of Big Data, along with the architectures used to store and process it.

## Quick Reference

| Term | Meaning |
|------|---------|
| Big Data | Data that is too large/fast/varied for traditional tools |
| Hadoop | Open-source framework for distributed storage (HDFS) and processing (MapReduce) |
| Spark | Fast, in-memory distributed computing engine |
| HDFS | Hadoop Distributed File System |
| MapReduce | Programming model: Map (transform) + Reduce (aggregate) |
| Data Warehouse | Structured storage optimised for BI reporting |
| Data Lake | Raw, unstructured storage at any scale |
| Kafka | Real-time streaming data platform |

    6.2 Supervised vs. Unsupervised Learning¶

    Theory¶

    Supervised Learning¶

    In supervised learning, the model is trained on a labelled dataset — each training sample \((x_i, y_i)\) has an input \(x_i\) and a known correct output \(y_i\). The model learns to map inputs to outputs so it can predict \(y\) for new, unseen \(x\) values.

    The learning process:

    \[ \underbrace{f(X)}_{\text{learned function}} \approx Y, \quad \text{minimising loss } \mathcal{L}(\hat{y}, y) \]

    Unsupervised Learning¶

    In unsupervised learning, the model is trained on unlabelled data — only inputs \(x_i\) are provided with no corresponding outputs. The model must discover structure on its own.

    Side-by-Side Comparison¶

    Aspect Supervised Unsupervised
    Labels required Yes No
    Goal Predict output Discover structure
    Output Specific label or value Cluster, component, rule
    Evaluation Accuracy, RMSE, F1 Silhouette score, Inertia
    Data cost High (labelling is expensive) Low (no labelling needed)
    Human oversight High Low
    Examples Linear Regression, SVM, NN K-Means, PCA, Apriori
    Typical use Fraud detection, spam filter Customer segmentation, anomaly detection

    Key Supervised Learning Algorithms¶

    Algorithm Task Strength
    Linear Regression Regression Simple, interpretable
    Logistic Regression Classification Probabilistic output
    Decision Tree Both Interpretable, handles non-linearity
    Random Forest Both High accuracy, handles overfitting
    SVM Both Effective in high dimensions
    Neural Networks Both Very powerful for complex patterns
    K-Nearest Neighbours Both No training phase

    Key Unsupervised Learning Algorithms¶

    Algorithm Task Strength
    K-Means Clustering Fast, scalable
    DBSCAN Clustering Handles noise, arbitrary shapes
    Hierarchical Clustering No need to specify k
    PCA Dimensionality Reduction Decorrelates features
    Autoencoders Both Non-linear dimensionality reduction
    Apriori Association Rules Market basket analysis

    # Program : Supervised vs. Unsupervised Learning Comparison
# Topic   : 6.2 Supervised vs. Unsupervised Learning
# Author  : BT255CO Lecture Notes

import numpy as np
import pandas as pd
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.cluster import KMeans
from sklearn.metrics import accuracy_score, silhouette_score
from sklearn.preprocessing import StandardScaler

# Use Iris dataset (a classic ML benchmark)
iris = load_iris()
X    = iris.data          # 4 features: sepal length, sepal width, petal length, petal width
y    = iris.target        # 3 classes: setosa, versicolor, virginica

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

print("Iris Dataset: 150 samples, 4 features, 3 classes")
print(f"Classes: {iris.target_names.tolist()}\n")

# =========================================================
# SUPERVISED: Logistic Regression
# (Uses labels y)
# =========================================================
print("=" * 55)
print("SUPERVISED: Logistic Regression")
print("=" * 55)

X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y, test_size=0.3, random_state=42, stratify=y
)

lr = LogisticRegression(max_iter=300)
lr.fit(X_train, y_train)
y_pred = lr.predict(X_test)
sup_acc = accuracy_score(y_test, y_pred)

print(f"Training samples : {len(X_train)}")
print(f"Test samples     : {len(X_test)}")
print(f"Test Accuracy    : {sup_acc * 100:.1f}%")
print(f"Advantages: Uses labels → precise; evaluable with accuracy\n")

# =========================================================
# UNSUPERVISED: K-Means Clustering
# (Does NOT use labels y)
# =========================================================
print("=" * 55)
print("UNSUPERVISED: K-Means (k=3)")
print("=" * 55)

kmeans = KMeans(n_clusters=3, random_state=42, n_init=10)
cluster_labels = kmeans.fit_predict(X_scaled)

sil_score = silhouette_score(X_scaled, cluster_labels)
inertia   = kmeans.inertia_

print(f"Cluster 0 count: {(cluster_labels == 0).sum()}")
print(f"Cluster 1 count: {(cluster_labels == 1).sum()}")
print(f"Cluster 2 count: {(cluster_labels == 2).sum()}")
print(f"Inertia         : {inertia:.2f}")
print(f"Silhouette Score: {sil_score:.3f}  (1=perfect, 0=overlap, -1=wrong)")
print(f"Advantages: No labels needed → works on any data\n")

# =========================================================
# Compare: how well do clusters align with true labels?
# =========================================================
print("=" * 55)
print("CLUSTER vs. TRUE LABEL ALIGNMENT")
print("=" * 55)
df_compare = pd.DataFrame({
    "true_label":    [iris.target_names[i] for i in y],
    "cluster":       cluster_labels
})
print(pd.crosstab(df_compare["cluster"], df_compare["true_label"]))
print("\n→ Cluster 0 ≈ setosa, Cluster 1 ≈ versicolor, Cluster 2 ≈ virginica")
print("  (without ever seeing the labels!)")

    Summary¶

    Key Takeaways

    • Supervised needs labels, makes predictions; Unsupervised discovers patterns without labels
    • Supervised learning is directly evaluable with accuracy, RMSE, etc.
    • Unsupervised evaluation uses indirect metrics: Silhouette Score, Inertia, Elbow method
    • K-Means on Iris (unlabelled) naturally rediscovers the three species — a powerful demonstration of unsupervised learning
    • Most real-world pipelines use both — unsupervised for EDA and feature engineering, supervised for final prediction
